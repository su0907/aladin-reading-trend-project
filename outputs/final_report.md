알라딘 월간 베스트셀러 데이터를 기반으로 독서 트렌드 변화를 분석하고, 주요 문화적 사건(한강 노벨문학상 수상, 더 퍼스트 슬램덩크 아카데미상 수상)이 베스트셀러 출판 시장에 미친 영향을 심층 분석한 프로젝트의 최종 보고서입니다.

# 2. 데이터 수집

## 1단계 크롤링 (월간 베스트셀러 리스트)
- 수집 대상: 알라딘 월간 베스트셀러 TOP 50 (2020.01~2025.11, 71개월)
- 도구: BeautifulSoup4, urllib
- 수집 항목:
  - year / month / rank  
  - category (상위 카테고리)
  - title, author  
  - price, star_score  
  - item_id (도서 고유 ID)
- 수집량: 
  - 이론적 수집량: 71개월 × 50개 = 3,550개
  - 실제 수집량: 3,539개
  - 차이: 11개 (크롤링 중 일부 데이터 누락)

## 2단계 크롤링 (도서 상세 페이지)
- 수집 대상: 고유 도서 1,960개
- 도구: BeautifulSoup4 + 병렬 처리
- 수집 항목:
  - real_category (상세 카테고리)
  - page_count (페이지 수)

## 크롤링 이슈 및 해결

### 문제 1: HTML 선택자 오타
```
초기 코드: category_tag = item.select_one("span.tit_catrgory")  # 오타
수정 코드: category_tag = item.select_one("span.tit_category")  # 정상

결과:
- 오타로 인해 모든 도서 카테고리 "N/A" 발생
- 수정 후 "국내도서" 포괄 카테고리 정상 수집
- 데이터 완성도 향상
```

### 문제 2: 성인 도서 크롤링 제한
```
성인 인증이 필요한 도서:
- 세부 카테고리 접근 불가
- 상세 페이지 접근 불가
- page_count = 0으로 수집됨
- 전처리 단계에서 제거 (22개)
```

### 문제 3: 특정 월 순위 50위 크롤링 실패
```
영향받은 월:
- 2020년 4월: 1개 누락 (49개 수집)
- 2023년 2~11월: 각 1개씩 누락 (각 49개 수집, 10개월)
총 누락: 11개

원인:
- BeautifulSoup으로 HTML 파싱 시 50위 항목이 선택되지 않음
- soup.select("div.ss_book_box") 결과가 49개만 반환
- 알라딘 페이지에는 50위가 표시되지만 동적 로딩 또는 HTML 구조 차이로 인한 것으로 추정
- Selenium을 통한 재크롤링 시도했으나 Colab 환경 제약으로 실패

해결 시도:
- Selenium을 사용한 동적 페이지 크롤링 시도
- JavaScript 렌더링 대기 시간 추가
- Colab 환경 문제로 최종 실패

결론:
- 11개 데이터는 수집하지 못함
- 전체 데이터의 0.3% 수준으로 분석에 미치는 영향 미미
```

### 시각화 1 검증 2: 연도별 평균 가격 추이 (장르별 비교)

![price_by_genre](../data/visualizations/01_2_price_verification_by_genre.png)

#### 주요 데이터
```
만화/어린이 제외 일반 도서:
- 2020년: 15,300원
- 2021년: 16,400원
- 2022년: 16,000원
- 2023년: 17,100원
- 2024년: 17,000원
- 2025년: 16,900원

만화:
- 2020년: 11,000원
- 2025년: 10,200원
- 변화: -7.3% 하락

어린이:
- 2020년: 12,200원
- 2025년: 15,700원
- 변화: +28.7% 상승
```

#### 인사이트: 2022년 가격 하락의 구조적 원인

**1. 만화/어린이 제외 일반 도서는 지속적 상승세 유지**
- 2020년 15,300원 → 2025년 16,900원 (+10.5%)
- 2022년에도 16,000원으로 2020년(15,300원) 대비 +4.6% 높은 수준
- 일반 도서만 보면 2022년 하락이 없었음

**2. 만화 장르의 저가 특성**
- 평균 가격: 약 8,300~11,000원 (일반 도서 대비 약 40~50% 수준)
- 2022년 만화 진입 90회 (2021년 73회 대비 +23% 증가)
- 저가 장르 비중 증가가 전체 평균을 하향

**3. 2022년 전체 평균 하락의 메커니즘**
```
2021년:
- 일반 도서: 16,400원 (높은 가격대)
- 만화·어린이: 140회 진입
- 전체 평균: 15,386원

2022년:
- 일반 도서: 16,000원 (여전히 높은 가격대)
- 만화·어린이: 157회 진입 (+17회, +12%)
- 저가 장르 비중 증가 → 전체 평균: 14,481원 (-5.9%)
```

**4. 시각화 1, 2와의 연관성**
- **가격 하락(-5.9%)**: 저가 장르(만화·어린이) 비중 증가가 직접 원인
- **페이지 수 하락(-14.0%)**: 만화·어린이는 평균 페이지가 적음
- **일반 도서는 정상**: 만화/어린이를 제외하면 가격·페이지 모두 상승세 유지
- **장르 구성의 영향**: 베스트셀러 평균 지표는 장르 구성에 크게 좌우됨

**5. 결론**
2022년 베스트셀러 평균 가격 하락은 출판 시장 전반의 가격 하락이 아니라, 저가·저페이지 장르(만화·어린이)의 베스트셀러 진입 증가로 인한 **구조적 변화**입니다. 이는 동일 시기 전체 신간 평균 가격이 상승(+4.4%)한 것과 대조되며, 베스트셀러와 전체 출판 시장의 특성 차이를 명확히 보여줍니다.

![price_comparison_chart](../data/visualizations/references/2022_book_price.jpeg)

*출처: 대한출판문화협회(2023.07.17)*

| 구분 | 2021년 | 2022년 | 증감 |
|------|--------|--------|------|
| 베스트셀러 평균 (본 연구) | 15,386원 | 14,481원 | -5.9% ↓ |
| 전체 신간 평균 (대한출판문화협회) | 17,116원 | 17,869원 | +4.4% ↑ |

**차이 발생 원인:**
- 베스트셀러와 전체 신간의 평균 가격이 상반된 방향으로 변화
- 베스트셀러: 2022년 만화·어린이 등 저가 장르 비중 증가로 평균 가격 하락
- 전체 신간: 지속적인 가격 상승
- 베스트셀러 평균 가격은 해당 해의 장르 구성에 따라 변동

이는 베스트셀러 분석이 전체 출판 시장과 다른 특성을 보일 수 있음을 나타냅니다.

### 2. 플랫폼 한계
- 알라딘 단일 플랫폼만 분석
- 교보문고, YES24 등 다른 서점 데이터 미포함
- 온라인 전용 판매, 독립서점 데이터 미반영

### 3. 정량 데이터 중심 분석
- 베스트셀러 진입 횟수, 가격, 평점 등 정량 데이터 중심
- 독자 구매 동기, 선호 이유 등 정성적 데이터 미포함
- 설문조사, 인터뷰 등 추가 데이터 수집 필요
