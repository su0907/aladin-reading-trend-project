# 📘 알라딘 월간 베스트셀러 독서 트렌드 분석 보고서  
**기간: 2020년 1월 ~ 2025년 11월**

알라딘 월간 베스트셀러 데이터를 기반으로 독서 트렌드 변화를 분석하고, GPT 예측 결과와 실제 데이터를 비교한 프로젝트의 최종 보고서입니다.

---

# 1. 프로젝트 목적

본 프로젝트는 2020~2025년 알라딘 월간 베스트셀러 데이터를 크롤링 및 분석하여 다음을 파악하는 것을 목표로 합니다.

- **연도별 독서 트렌드 변화 분석**
- **카테고리별 인기 장르 변화**
- **GPT 예측 결과와 실제 데이터 비교**
- **사회적 사건(코로나, AI 시대 등)과 독서 트렌드의 상관성 분석**

---

# 2. 데이터 수집

### **1단계 크롤링**
- 알라딘 월간 베스트셀러 TOP 50 (2020.01~2025.11)
- BeautifulSoup 사용
- 수집 항목:
  - year / month / rank  
  - category (상위 카테고리, 예: 국내도서)  
  - title, author  
  - price, star_score  
  - item_id

### **2단계 상세 페이지 크롤링**
- 도서 상세 페이지 조회 (item_id 기반)
- 수집 항목:
  - real_category (세부 카테고리, 예: 인문학/교양)
  - page_count

### 크롤링 이슈 & 해결
- 알라딘은 원래 `tit_category` 태그를 사용 → **정상적으로 ‘국내도서’ 수집됨**
- 초기에 코드 오타(`tit_catrgory`)로 일부 카테고리가 N/A 처리되었으나 수정 후 정상 수집됨
- 성인 도서는 페이지 수가 제공되지 않아 지속적으로 `page_count = 0` → **모두 삭제(전처리 단계)**

# 3. 데이터 저장 & 구조화

- 1단계 데이터: `data/raw/aladin.csv`

# 4. 데이터 전처리

### 주요 전처리 작업

- item_id 기준 병합
- real_category가 존재하면 category로 업데이트
- 페이지 수 0인 도서 삭제 (주로 성인도서)
- 결측치 제거 및 형식 통일

- 단기적인 피크 이후 완만한 감소 → 매체 이슈 기반의 일시적 관심 증가로 해석

# 6. GPT 예측 vs 실제 비교

| GPT 예측 | 실제 데이터 결과 |
|---------|------------------|
| 코로나 시기: 에세이·아동·웹툰 증가 | 완벽하게 일치 |
| 포스트 코로나: 경제·자기계발 증가 | 일치 |
| 2024~2025: AI·기술 서적 증가 | 실제로 등장 |
| SNS 영향으로 단기 급등 도서 증가 | 월간 1위 변동성에서 확인됨 |

→ **GPT 예측 정확도 매우 높음**
