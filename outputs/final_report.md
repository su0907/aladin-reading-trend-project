# 2. 데이터 수집

## 1단계 크롤링 (월간 베스트셀러 리스트)
- 수집 대상: 알라딘 월간 베스트셀러 TOP 50 (2020.01~2025.11, 71개월)
- 도구: BeautifulSoup4, urllib

  - category (상위 카테고리)
  - title, author

- 수집량: 
  - 이론적 수집량: 71개월 × 50개 = 3,550개
  - 실제 수집량: 3,539개
  - 차이: 11개 (크롤링 중 일부 데이터 누락)

## 2단계 크롤링 (도서 상세 페이지)
- 수집 대상: 고유 도서 1,960개
- 도구: BeautifulSoup4 + 병렬 처리
- 수집 항목:
  - real_category (상세 카테고리)
  - page_count (페이지 수)

## 크롤링 이슈 및 해결

### 문제 1: HTML 선택자 오타
```

### 문제 3: 특정 월 순위 50위 크롤링 실패
```

- 알라딘 페이지에는 50위가 표시되지만 동적 로딩 또는 HTML 구조 차이로 인한 것으로 추정
- Selenium을 통한 재크롤링 시도했으나 Colab 환경 제약으로 실패

해결 시도:
- Selenium을 사용한 동적 페이지 크롤링 시도
- JavaScript 렌더링 대기 시간 추가
- Colab 환경 문제로 최종 실패

결론:
- 11개 데이터는 수집하지 못함
- 전체 데이터의 0.3% 수준으로 분석에 미치는 영향 미미

### 시각화 1 검증 2: 연도별 평균 가격 추이 (장르별 비교)

![price_by_genre](../data/visualizations/01_2_price_verification_by_genre.png)

#### 주요 데이터
```
만화/어린이 제외 일반 도서:
- 2020년: 15,300원
- 2021년: 16,400원
- 2022년: 16,000원
- 2023년: 17,100원
- 2024년: 17,000원
- 2025년: 16,900원

만화:
- 2020년: 11,000원
- 2025년: 10,200원
- 변화: -7.3% 하락

어린이:
- 2020년: 12,200원
- 2025년: 15,700원
- 변화: +28.7% 상승
```

#### 인사이트: 2022년 가격 하락의 구조적 원인

**1. 만화/어린이 제외 베스트셀러도 소폭 하락**
- 2020년 15,300원 → 2025년 16,900원 (+10.5%)
- 2021년 16,400원 → 2022년 16,000원 (-2.4% 하락)
- 2022년에도 소폭 하락했지만, 전체 평균(-5.9%)보다는 작은 폭

**2. 만화 장르의 저가 특성**
- 평균 가격: 약 8,300~11,000원
- 일반 도서 대비: 약 50~72% 수준 (연도별 차이 있음)
- 2022년 기준 약 52% 수준으로 가장 낮음
- 2022년 만화 진입 90회 (2021년 73회 대비 +23% 증가)
- 저가 장르 비중 증가가 전체 평균을 추가로 하향시킨 것으로 분석됨

**3. 2022년 전체 평균 하락의 메커니즘**
```
2021년:
- 만화/어린이 제외: 16,400원
- 만화·어린이: 140회 진입
- 전체 평균: 15,386원

2022년:
- 만화/어린이 제외: 16,000원 (-2.4%)
- 만화·어린이: 157회 진입 (+17회, +12%)
- 이중 효과:
  1. 만화/어린이 제외 베스트셀러 자체도 소폭 하락 (-400원)
  2. 저가 장르 비중 증가로 추가 하락
- 결과: 전체 평균 14,481원 (-5.9%)
```

**4. 메인 시각화와의 연관성**
- **가격 하락(-5.9%)**: 
  - 만화/어린이 제외 베스트셀러 자체 하락 (-2.4%)
  - 저가 장르(만화·어린이) 비중 증가 (추가 -3.5%p)
- **페이지 수 하락(-14.0%)**: 만화·어린이는 평균 페이지가 적음
- **복합적 원인**: 만화/어린이 제외 베스트셀러 하락 + 장르 구성 변화
- **장르 구성의 영향**: 베스트셀러 평균 지표는 장르 구성에 크게 영향받음

# 6. GPT 예측 vs 실제 비교

## 연구 배경

본 프로젝트의 목적은 GPT 예측의 정확도를 평가하는 것이 아니라, **예측 모델과 실제 데이터 간 차이를 비교함으로써 데이터 기반 실증 분석의 중요성을 확인하는 데 있습니다.** AI 예측 모델이 포착할 수 있는 패턴과 예측하지 못하는 외부 변수를 구분하여, 실제 데이터 분석의 필요성을 제시하고자 합니다.

## GPT 예측 내용

### 2020~2021년 (코로나 초기)
- 예측: 아동·웹툰·에세이 강세, 힐링 도서 증가
- 실제: 부분 적중 - 만화 453개(2위), 어린이 289개(5위)

### 2022~2023년 (포스트 코로나)
- 예측: 경제·자기계발 증가, 재테크 붐
- 실제: 적중 - 경제경영 2021년 20% 정점

### 2024~2025년 (AI 시대)
- 예측: AI·IT 서적 강세, 실무 실용서 증가
- 실제: 미적중 - 인문학 급증(3배), AI/IT 서적 TOP 20에 없음

## GPT 예측 평가

### 맞춘 트렌드
```
1. 코로나 시기 만화·아동 강세
2. 경제경영 증가 (2021-2023)
3. 재테크 관련 도서 붐
4. 웹툰·아동 꾸준한 인기
```

### 예측하지 못한 트렌드
```
1. 한강 노벨문학상 효과 (2024년 최대 이슈)
   - 전혀 예측하지 못함
   - 757% 증가, 시장 점유율 0.5% → 7.2%

2. 인문학의 폭발적 성장
   - 2020년 8% → 2025년 23% (약 3배)
   - GPT는 "AI 서적"만 예측

3. AI/IT 서적의 부재
   - 예측과 달리 TOP 20에 나타나지 않음
   - 오히려 인문학이 급증
```

## 예측의 한계

### GPT가 놓친 핵심 이유

1. **예측 불가능한 외부 사건**
   - 한강 노벨문학상 (2024.10)
   - 사회적 이벤트는 AI 예측 범위 밖

2. **트렌드의 질적 전환 간과**
   - "정보 소비 → 성찰" 전환
   - 인문학 급증 예측 실패

3. **과대 예측**
   - AI/IT 서적 진입 예상
   - 실제로는 나타나지 않음

## 시사점

GPT 예측은 과거 패턴 기반 트렌드(코로나 → 경제불안 → 실용서 증가)는 잘 포착했으나, 예상치 못한 외부 요인(노벨상 수상)과 인문학 성장 같은 구조적 변화는 포착하지 못한 것으로 나타났습니다. 이는 AI 예측의 한계를 보여주며, 실제 데이터 분석과 실시간 모니터링을 병행해야 함을 시사합니다.
