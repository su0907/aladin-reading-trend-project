# 알라딘 월간 베스트셀러 독서 트렌드 분석 보고서  
**기간: 2020년 1월 ~ 2025년 11월**

알라딘 월간 베스트셀러 데이터를 기반으로 독서 트렌드 변화를 분석하고, 주요 문화적 사건(한강 노벨문학상 수상, 더 퍼스트 슬램덩크 아카데미상 수상)이 베스트셀러 출판 시장에 미친 영향을 심층 분석한 프로젝트의 최종 보고서입니다.

---

# 1. 프로젝트 목적

본 프로젝트는 2020~2025년 알라딘 월간 베스트셀러 데이터를 크롤링 및 분석하여 다음을 파악하는 것을 목표로 합니다.

- 연도별 독서 트렌드 변화 분석
- 카테고리별 인기 장르 변화
- 한강 노벨문학상 효과 심층 분석 (핵심)
- GPT 예측 결과와 실제 데이터 비교
- 사회적 사건과 독서 트렌드의 상관성 분석

---

# 2. 데이터 수집

## 1단계 크롤링 (월간 베스트셀러 리스트)
- 수집 대상: 알라딘 월간 베스트셀러 TOP 50 (2020.01~2025.11, 71개월)
- 도구: BeautifulSoup4, urllib
- 수집 항목:
  - year / month / rank  
  - category (상위 카테고리)
  - title, author  
  - price, star_score  
  - item_id (도서 고유 ID)
- 수집량: 
  - 이론적 수집량: 71개월 × 50개 = 3,550개
  - 실제 수집량: 3,539개
  - 차이: 11개 (크롤링 중 일부 데이터 누락)

## 2단계 크롤링 (도서 상세 페이지)
- 수집 대상: 고유 도서 1,960개
- 도구: BeautifulSoup4 + 병렬 처리
- 수집 항목:
  - real_category (상세 카테고리)
  - page_count (페이지 수)
