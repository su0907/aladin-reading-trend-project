# 📘 알라딘 월간 베스트셀러 독서 트렌드 분석 보고서  
**기간: 2020년 1월 ~ 2025년 11월**

알라딘 월간 베스트셀러 데이터를 기반으로 독서 트렌드 변화를 분석하고, GPT 예측 결과와 실제 데이터를 비교한 프로젝트의 최종 보고서입니다.

---

# 1. 프로젝트 목적

본 프로젝트는 2020~2025년 알라딘 월간 베스트셀러 데이터를 크롤링 및 분석하여 다음을 파악하는 것을 목표로 합니다.

- **연도별 독서 트렌드 변화 분석**
- **카테고리별 인기 장르 변화**
- **GPT 예측 결과와 실제 데이터 비교**
- **사회적 사건(코로나, AI 시대 등)과 독서 트렌드의 상관성 분석**

---

# 2. 데이터 수집

## ✅ 1단계 크롤링
- 수집 대상: 알라딘 월간 베스트셀러 TOP 50 (2020.01~2025.11)
- BeautifulSoup 사용
- 수집 항목:
  - year / month / rank  
  - category (상위 카테고리, 예: 국내도서)  
  - title, author  
  - price, star_score  
  - item_id

## ✅ 2단계 상세 페이지 크롤링
- 도서 상세 페이지 (item_id 기반)
- 수집 항목:
  - real_category (세부 카테고리)
  - page_count

## 🔧 크롤링 이슈 & 해결
- 알라딘은 원래 `tit_category` 태그를 사용 → **정상적으로 ‘국내도서’ 수집**
- 초기에 코드 오타(`tit_catrgory`)로 일부 카테고리 N/A 발생 → 수정 후 정상 수집
- 성인 도서는 상세 페이지에서 page_count가 제공되지 않아 **page_count = 0 처리됨**  
  → 전처리 단계에서 삭제

- 사회적 사건이 독서 트렌드에 빠르게 반영됨  
- 2020~2025 트렌드는
